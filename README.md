# Carggo

## Installation

    $ gem install carggo

## Usage

    $ carggo find postgres #=> postgres = 0.16.0-rc.1

    $ carggo add postgres rand ... #=> go to your Cargo.toml file :))

    $ carggo remove postgres rand ... #=> go to your Cargo.toml file again :))
